# Gemini ICPC 2025 Submissions

This repository contains the code submissions from an advanced version of
[Gemini 2.5 Deep Think](https://blog.google/products/gemini/gemini-2-5-deep-think/)
for the 2025 International Collegiate Programming Contest World Finals.
